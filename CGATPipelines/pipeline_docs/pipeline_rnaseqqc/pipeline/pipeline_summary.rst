=================
Pipeline Summary
=================

Aims of page
============

This page summarises the structure of the rnaseqqc pipeline used to perform the 
qc of rna-seq data and form this report. Detailed on this page are:

  * The errors and warnings generated whilst running the report

  * A diagram of the pipeline detailing the workflow 

  * The pipeline documentation detailing the inputs, outputs and purpose of each
	function
